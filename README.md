# CSS Advanced Project

This project is part of the **ALX Front-End Curriculum**, focused on implementing advanced CSS styling based on a provided Figma design.

## 📌 Objective

To build and style a multi-section responsive website using **semantic HTML** and **modern CSS** techniques (Flexbox, Grid, and custom styling), while adhering strictly to design guidelines.

## 🧱 Project Structure
lx_html_css/
└── css_advanced/
├── index.html
├── styles.css
├── images/
└── README.md

## 🎨 Design Source

- UI/UX Design provided via Figma
- Fonts used: Source Sans Pro, Spin-Cycle-OT
- Downloaded image assets placed inside the `images/` directory

## 🧩 Sections Implemented

1. **Header and Banner** — Navigation, Hero Text, Call-to-Action, Profile icons
2. **Quote Section** — Testimonial block with avatar
3. **Video Tutorials** — Grid layout of video cards
4. **Membership Features** — Benefits of signing up
5. **FAQ Section** — Common questions in two-column layout
6. **Footer** — Branding, social media links, copyright

## 🛠️ Technologies Used

- HTML5 (Semantic)
- CSS3 (Flexbox, Grid, custom properties)
- No frameworks or libraries (vanilla HTML/CSS only)
- Fully W3C validated
